# Exporting official Foundry examples

To make the compiler generate truly native documents, export working examples from Foundry.

Recommended examples:

## Spells

- Fog Cloud
- Fireball
- Bless
- Mage Armor
- Haste
- Shield of Faith
- Invisibility

## Classes

- Wizard
- Barbarian
- Cleric
- Warlock
- Paladin

## Species/Races

- Human
- Elf
- Dwarf

## Items

- Longsword
- Shield
- Bag of Holding
- Wand of Magic Missiles

In Foundry:

1. Open a compendium document.
2. Right click or use the top-right options menu.
3. Choose **Export Data**.
4. Save the JSON into `data/official_examples/`.

The compiler can use those examples as templates and duplicate filters.
