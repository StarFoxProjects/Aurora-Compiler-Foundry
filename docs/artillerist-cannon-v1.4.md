# Artillerist Eldritch Cannon v1.4

The Artillerist cannon options are now merged into the single **Eldritch Cannon** feature.

The compiler no longer grants **Flamethrower**, **Force Ballista**, and **Protector** as separate actor features when **Eldritch Cannon** is granted. Instead, the single feature contains four activities:

- Create Eldritch Cannon — dnd5e summon activity, places an Eldritch Cannon token from the module summon actor pack.
- Flamethrower — bonus action Dex save, 15 ft cone, 2d8 fire.
- Force Ballista — bonus action ranged spell attack, 120 ft, 2d8 force.
- Protector — bonus action temporary HP, 10 ft radius, 1d8 + INT modifier.

The summoned token uses the dnd5e system summon activity. Players may need permission to create summoned tokens, depending on the world's dnd5e settings.
